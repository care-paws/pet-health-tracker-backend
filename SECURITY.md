# Security Policy  
  
## Supported Versions  
| Version | Supported |  
|---------|-----------|  
| 0.1.x   | ✅        |  
  
## Reporting a Vulnerability  
  
Para reportar vulnerabilidades de seguridad:  
- Correo: raraulalmao7@gmail.com (Si es miembro del equipo, a traves del canal de Discord)
- Tiempo de respuesta inicial: inmediata  
- Proceso: Evaluación → Parche → Comunicación  
  
## Security Overview  
  
Implementamos prácticas de seguridad estándar de la industria:  
- Validación de entrada en todas las capas  
- Autenticación requerida para endpoints sensibles  
- Manejo controlado de errores  
- Base de datos segura con ORM   
  
## Known Limitations  
  
Este es un proyecto MVP en desarrollo activo. Algunas características de seguridad están en progreso y se documentarán en futuras actualizaciones.  
  
## Contact  
  
Para cuestiones de seguridad no críticas, use GitHub Issues. Para vulnerabilidades de seguridad, use el correo de reporte mencionado anteriormente.
